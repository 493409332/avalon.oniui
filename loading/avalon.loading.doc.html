loading组件，

使用说明
名字默认值说明
配置参数
timer 160 毫秒数，动画效果帧间隔
type "ball" 类型，默认是ball，球，可以配置成 {"type": "ball", "widthInner": 3} ...
toggle true 是否显示
modal true 是否显示遮罩
opacity 0.1 遮罩透明度
container void 0 loading效果显示的容器，默认是绑定widget的元素
onInit回调函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
getTemplate配置接口getTemplate(tpl, opts, tplName) 定制修改模板接口
showLoading接口调用showLoading() 显示loading效果
hideLoading接口调用hideLoading() 隐藏loading
destroyLoading接口调用destroyLoading() 销毁loading
