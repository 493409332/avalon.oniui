名字默认值 说明
onText "ON" 选中状态提示文字
offText "OFF" 未选中状态提示文字
type "normal" 滑动条类型，默认normal，可设置为large,small,mini，以及其他任意组件不自带的名词，可以用来注入自定义class，生成ui-switch-{{type}}添加给switch模板容器
theme "normal" 主题，normal,success,warning,danger
draggable false 是否支持拖动切换状态
disabled false 禁用
checked false 默认选中状态
animated true 是否开启切换动画效果
hdir "on-off" 开启、关闭选项排列顺序on-off,off-on
$vmDragging false 为true，不响应click切换效果
$animateArrMaker配置函数$animateArrMaker(from, to) 不支持css3动画效果步长生成器函数，返回一个数组，类似[0,xx,xx,xx,50]
onChange配置函数onChange(newValue, vmodel) 选中状态发生变化时触发，参数为当前的选中状态及vmodel对象
getTemplate配置函数getTemplate(tmp, opts) 用于修改模板的接口，默认不做修改
toggle接口函数toggle 交替改变选中状态
disable接口函数disable 禁用组件
enable接口函数enable 启用组件

switch 组件有一个重要的xx方法,见源码：
  1. 不同大小type，默认设置，选中状态
  2. 不同大小type，支持可拖动切换，选中状态
  3. 不同大小type，OFF-ON顺序排列，选中状态
  4. 不同大小type，disabled状态，选中状态
  5. 不同大小type，自定义ON-OFF文本，选中状态
  6. 不同大小type，通过接口修改状态，选中状态
