menu使用说明

menu组件，实现扫描dom元素或者设置传参生成级联菜单的组件，注意扫描dom情形下，会销毁原有的dom，且会忽略所有的ol，ul，li元素上原有的绑定

menu组件配置项说明：
名字默认值说明
activefalse 将第几个项目设置为选中，级联情形下，会将设置应用给每一级menu，默认是false，一个都不选中
_avtive[]
event "mouseenter" 选中事件，默认mouseenter
dir "h" 方向，取值v,h，默认h是水平方向， v是竖直方向
onInit配置函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
resetMenu配置函数resetMenu(vmodel) 重置menu的配置方法，默认是重置为一个都不选中
getTemplate配置函数getTemplate(tpl, opts, tplName) 定制修改模板接口
onActivate配置函数onActivate(event, index) 选中menu后的回调，this指向对应的menu li元素，参数是事件对象，索引 fn(event, index)，默认为avalon.noop
onClickActive配置函数onClickActive(event, index) 点击选中的menu，this指向对应的menu li元素，参数是事件对象，索引 fn(event, index)，默认为avalon.noop
activate接口函数activate(index)展开菜单索引为index的项目，index置为false,undefined则不会展开任一项目
apiName接口函数apiName(argx) description
