名字默认值 说明
position "right" scrollbar出现的位置,right右侧，bottom下侧，可能同时出现多个方向滚动条
limitRateV 1.5 竖直方向，拖动头最小高度和拖动头宽度比率
limitRateH 1.5 水平方向，拖动头最小宽度和高度的比率
scrollTop 0 竖直方向滚动初始值，负数会被当成0，设置一个极大值等价于将拖动头置于bottom
scrollLeft 0 水平方向滚动初始值，负数会被当成0处理，极大值等价于拖动头置于right
show "always" never一直不可见，scrolling滚动和hover时候可见，always一直可见
showBarHeader true 是否显示滚动条两端的上下箭头
draggerHTML "" 滚动条拖动头里，注入的html碎片
size "normal" srollbar size,normal为10px，small为8px，large为14px
onInit配置函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
getTemplate配置函数getTemplate(tpl, opts) 定制修改模板接口
getBars接口函数getBars()返回所有的滚动条元素
getScroller接口函数getScroller()返回scroller
update接口函数update()更新滚动条状态，windowresize，内容高度变化等情况下调用，不能带参数
scrollTo接口函数scrollTo(x,y) 滚动至 x,y

scrollbar 组件有一个重要的xx方法,见源码：
  1. scrollbar-document.body带上下小箭头
  2. scrollbar-元素带上下小箭头双滚动条
  3. scrollbar-带上下小箭头嵌套
  4. scrollbar-元素无上下小箭头双滚动条
  5. scrollbar-滚动条hover或者滚动时候可见，初始不可见
  6. scrollbar-滚动条不可见
