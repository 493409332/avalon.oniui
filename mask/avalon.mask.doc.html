mask

输入引导模块

这是ms-duplex2.0的一个扩展模块，用于引导用户输入。

通过如下方式使用:

<input ms-duplex-mask="a" data-duplex-mask="((00/00其他字符0000)" />
使用说明
名字 默认值 说明
配置参数
mask mask {String} 用于提示用户输入的mask字符串，用户只能在占位符上输入（会有光标引导你）,用户必须设置data-duplex-mask属性
element element {Element} 组件实例要作用的input元素
oldValue "" {String} 元素之前的value值
value "" {String} 元素现在的value值
impurity {} {Object} mask分别两部分，一些部分用户需要输入，一部分是提示或美化用的杂质，impurity是用于装载这些杂质在这个mask中的索引值，它用于光标引导功能
placehoder "_" {Boolean} "_", 将元字符串换为"_"显示到element.value上，如99/99/9999会替换为__/__/____，可以通过data-duplex-mask-placehoder设置
hideIfInvalid false {Boolean} false, 如果它不匹配就会在失去焦点时清空value，可以通过data-duplex-mask-hide-if-invalid设置
hideIfPristine true {Boolean} true如果它没有改动过就会在失去焦点时清空value，可以通过data-duplex-mask-hide-if-pristine设置
showIfHover false {Boolean} false 当鼠标掠过其元素上方就显示它出来，可以通过data-duplex-mask-show-if-hover设置
showIfFocus true {Boolean} true 当用户让其元素得到焦点就显示它出来，可以通过data-duplex-mask-show-if-focus设置
showAlways false {Boolean} false 总是显示它，可以通过data-duplex-mask-show-always设置
translations {Object} 此对象上每个键名都是元字符，都对应一个对象，上面有pattern(正则)，placehoder(占位符，如果你不想用"_"),optional（表示可选）
