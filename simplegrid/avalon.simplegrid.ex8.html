演示hover效果

要点

IE678下,需要通过getTemplate修改默认模板,为每一个tr加两个ms-class绑定,对应的类名自己在样式表添加

                        getTemplate: function(str) {
                            if (window.dispatchEvent) {
                                return str//IE678不支持div:hover效果
                            } else {
                                return str.replace('ms-repeat-row="_data"', 'ms-repeat-row="_data" ms-class-even="!($index % 2)" ms-class-odd="$index % 2"')
                            }
                        },

CSS

            .ui-grid-tbody tr.even td, .ui-grid-tbody tr.even td div{
                background: #f2f2f2;
            }
            .ui-grid-tbody tr.odd td, .ui-grid-tbody tr.odd td div{
                background:#ffe862;
            }
            .ui-grid-tbody tr:nth-child(even) td, .ui-grid-tbody tr:nth-child(even) td div{
                background: #f2f2f2;
            }
            .ui-grid-tbody tr:nth-child(odd) td, .ui-grid-tbody tr:nth-child(odd) td div{
                background: #ffe862;
            }
