名字默认值 说明
event "mouseenter" 显示tooltip的事件，默认hover的时候显示tooltip，为false的时候就不绑定事件，如果后面设置了自动隐藏，则mouseenter对应的是mouseleave,focus对应的是blur，进行自动隐藏事件侦听，使用代理的时候，目测不支持focus,blur
width "auto" tip宽度，默认是auto
height "auto" tip高度，默认是auto
arrow true 是否显示尖角图标，默认为true
autohide true 元素hoverout之后，是否自动隐藏tooltip，默认true
delegate false 元素是否只作为一个代理元素，这样适合对元素内多个子元素进行tooltip绑定
disabled false 禁用
track false tooltip是否跟随鼠标，默认否
animated true 是否开启显示隐藏切换动画效果
position "rt" tooltip相对于element的位置，like: "rt,rb,rc..."
positionMy false tooltip元素的定位点，like: "left top+11"
positionAt false element元素的定位点，like: "left top+11",positionAt && positionMy时候忽略position设置
hiddenDelay 64 tooltip自动隐藏时间，单位ms
onInit配置函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
contentGetter配置函数contentGetter() 获取内容接口，讲srcElement作为参数传递过来，默认是返回a标签的title，如果该函数返回为空，那么则不会显示tooltip
_animateArrMaker配置函数_animateArrMaker(from, to) 不支持css3动画效果步长生成器函数，返回一个数组，类似[0,xx,xx,xx,50]
getTemplate配置函数getTemplate(tpl, opts) 定制修改模板接口
show接口函数show(elem) 显示tooltip，相对于elem定位，elem为元素或者event事件对象，如果elem为空，则采用之前缓存的对象，两者都为空，则只展示，不改变位置
hide接口函数hide($event) 隐藏tooltip，参数是$event，可缺省

组件在不跟随鼠标移动时候，会自动将箭头对齐元素中间的位置，当tooltip没有元素高或者宽时，会自动调整tip显示在元素高度或者宽度居中的地方

tooltip 组件有一个重要的contentGetter方法,见源码：

           "contentGetter": function(elem) {
                if(elem.tagName.toLowerCase() != "a") return
                return elem.title
            }

该函数有无返回值，将决定是否展示tooltip

  1. input元素获取焦点显示tip
  2. input元素hover显示tip-父容器做代理
  3. 不使用代理，各个方向绑定，改变位置，点击展示
  4. 箭头位置自动调整，点击展示
  5. input元素获取焦点显示tip，不展示箭头
  6. 鼠标跟随
