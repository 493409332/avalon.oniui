名字默认值 说明
event "mouseenter" 显示tooltip的事件，默认hover的时候显示tooltip，为false的时候就不绑定事件
content "" tooltip显示内容，默认去获取element的title属性
autohide true 元素hoverout之后，是否自动隐藏tooltip，默认true
delegate false 元素是否只作为一个代理元素，这样适合对元素内多个子元素进行tooltip绑定
disabled false 禁用
track false tooltip是否跟随鼠标，默认否
animated true 是否开启显示隐藏切换动画效果
position "right+10 top" tooltip相对于element的位置，like: "(left|center|right)([+-]number)? (top|center|bottom)([+-]number)?"
hiddenDelay 16 tooltip自动隐藏时间
contentGetter配置函数contentGetter() 获取内容接口，讲srcElement作为参数传递过来，默认是返回title
$animateArrMaker配置函数$animateArrMaker(from, to) 不支持css3动画效果步长生成器函数，返回一个数组，类似[0,xx,xx,xx,50]
getTemplate配置函数getTemplate(tpl, opts) 定制修改模板接口
show接口函数show(elem) 显示tooltip，相对于elem定位，如果elem为空，则不改变位置
hide接口函数hide($event) 隐藏tooltip，参数是$event，可缺省
