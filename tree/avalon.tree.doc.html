使用说明
名字默认值说明
配置参数
view{Object} 视觉效果相关的配置
view.showLine true view.showLine是否显示连接线
view.dblClickExpand true view.dblClickExpand是否双击变化展开状态
view.selectedMulti true view.selectedMulti true / false 分别表示 支持 / 不支持 同时选中多个节点
view.showIcon true view.showIcon zTree 是否显示节点的图标
view.showTitle true view.showTitle 分别表示 显示 / 隐藏 提示信息
onInit回调函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
view.nameShower配置接口view.nameShower(leaf)节点显示内容过滤器，默认是显示leaf.name
getTemplate配置接口getTemplate(tpl, opts, tplName) 定制修改模板接口
exChangeNodes接口调用exChangeNodes(a, b, leaf, changeDom) 交换leaf节点的索引为a、b子节点的位置，doNotChangeDom不同步到dom，例如修改dom，同步数据而已
avalon.ui.tree.Extension接口调用avalon.ui.tree.Extension({xxx}) 给tree组件添加扩展，用于扩展属性，方法
