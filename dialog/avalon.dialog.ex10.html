通过加载avalon.draggable实现拖拽（ms-widget="dialog dialogId"）

说明：配置ms-widget="dialog,dialogId"就可实现一个简单的弹窗组件，组件默认有确定和取消按钮，点击任意按钮都会关闭弹窗，当然弹窗右上角也会显示关闭按钮，点击关闭弹窗，dialog的子元素可以是模板，组件会自动渲染

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>通过加载avalon.draggable实现拖拽</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset>
                            <legend>基本的dialog组件</legend>
                                <button ms-click="show('aa')">打开对话框</button>
                                <div ms-widget="dialog,aa">
                                    呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵
                                    <p>{{message}}</p>
                                </div>
                        </fieldset>
                        <script>
                            require(["dialog/avalon.dialog", "./draggable/avalon.draggable"], function(avalon) {
                                avalon.define("demo", function(vm) {
                                    vm.message = "笑傻了吧！！";
                                    vm.show = function( id ){
                                        avalon.vmodels[id].toggle = true;
                                    }
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
