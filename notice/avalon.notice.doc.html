notice使用文档

notice组件用来向用户显示一些提示信息

notice组件配置项说明：
name value description
container
type "info" 样式类型
content “” 要显示的内容DOM|String
header “” 标题
hasCloseBtn true 是否显示关闭
onHide avalon.noop 关闭回调
timer 0 自动消失时间
hasCloseBtn true 是否有关闭按钮
isPlace true 是否被下一个notice替换
isAffix false 是否吸顶
setContent
setType
toggle
setTitle
onShow

notice使用注意点

  1. 请保证输入域的值是数值或者字符串型的数值，否则会报错
  2. 当通过手动输入值时，如果输入的值不是数值型的，组件会默认将其置为修改之前的值，如果输入大于最大值，组件会自动将其置为最大值，小于最小值时同理
  3. 在输入域获得焦点时可通过键盘上的上下箭头控制notice的增减
  4. notice会判断初始输入域值是否在用户设置的数值范围呢，不在的话会进行调整
