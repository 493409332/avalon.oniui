miniswitch

下拉式开关 此组件基于switchdropdown
使用方法
名字默认值 说明
width 40 自定义宽度
listWidth 100 自定义下拉列表的宽度
height(new) 60 下拉列表的高度
enable(new) true 组件是否可用
readonly(new) false 组件是否只读
data [] 下拉列表显示的数据模型
value data中第一个数据项的value 设置组件的初始值
  • 用空select节点生成默认组件
  • 用select结构配置组件
  • 用options配置组件
  • 用options配置组件并设置duplex
