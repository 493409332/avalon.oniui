默认参数

dropdown组件配置项说明：
名字默认值 说明
width 200 自定义宽度
listWidth 200 自定义下拉列表的宽度
height 200 下拉列表的高度
enable true 组件是否可用
readonly false 组件是否只读
model [] 下拉列表显示的数据模型
modelBind false 是否要在model更新时通知列表更新
value model中第一个数据项的value 设置组件的初始值
multiple false 是否为多选模式
size 1 尺寸大小
