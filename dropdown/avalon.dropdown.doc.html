默认参数

名字默认值 说明
width 自定义宽度
listWidth 自定义下拉列表的宽度
height(new) 200 下拉列表的高度
enable(new) true 组件是否可用
readonly(new) false 组件是否只读
model [] 下拉列表显示的数据模型
textFiled(new) "text"||"name"(对上一版本进行兼容) 模型数据项中对应显示text的字段,可以传function，根据数据源对text值进行格式化
    function(item) {
        return item.text;
    }
valueField(new) "value" 模型数据项中对应value的字段
value model中第一个数据项的value 设置组件的初始值
label "" 设置组件的提示文案，可以是一个字符串，也可以是一个对象 
    {
        label: "This is a label..."
    }
或者是 
    {
        model: [],
        textField: 'text',
        valueField: 'value',
        label: {
            text: 'This is a label...',
            value: ''
        }
    }
titleTemplate title模板
optionTemplate 下拉菜单中每一个option的模板
listTemplate 下拉菜单模板
bodyTemplate 整体外围框架模板
