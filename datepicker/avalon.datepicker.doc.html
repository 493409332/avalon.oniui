notice使用文档

notice组件用来向用户显示一些提示信息

notice组件配置项说明：
name value description
altField 根据日历框的选择更新的输入域，使用altFormat格式化日期
altFormat
appendText 输入域后要显示的text，比如显示日期格式,示例：appendText:"(yyyy-mm-dd)"
autoSize 根据dataFormat自动resize输入域的大小
changeMonth false 是否可以通过下拉框选择月份
changeYear false 是否可以通过下拉框选择年份
dateFormat "mm/dd/yy" 显示日期的格式
dayNames [ "Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday" ]
defaultDate 显示日历框之后高亮的日期
firstDay 设置每一周的第一天是哪天，0代表Sunday，1代表Monday，依次类推
hideIfNoPrevNext false 当设置了minDate、maxDate时可能next、prev就失效禁用，如果设置了hideIfNoPrevNext为true那么组件会将其隐藏
maxDate 最大的可选日期，设为null，可以任意选
minDate 最小的可选日期，设为null则任选
nextText "next" "下个月"链接的显示文本
numberOfMonths 1 一次显示的日历数
onChangeMonthYear 当month或者year更新时调用的回调。参数为year、month、日历组件实例
onSelect
onClose
prevText
showButtonPanelType 是否在日历下显示button面板
stepMonths 当点击next、prev链接时应该跳过几个月份
yearRange 在year下拉框中显示的year范围，可以是相对于今年的格式"-nn:+nn",也可以是相对于当前选中日期year的范围"c-nn:c+nn",也可以是绝对的"nnnn:nnnn"
