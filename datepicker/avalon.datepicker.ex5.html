datepicker demo

toggle切换日期的显示隐藏

说明：可以通过toggle切换datepicker的显示与隐藏，这需要在绑定datepicker时，设置ms-data-toggle属性指向demo controller中的属性

可以通过配置timer来使组件可选择时间

calendarLabel配置日历的说明label

说明：配置calendarLabel来显示你独特的日历说明

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>datepicker demo</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset style="position:relative;">
                            <legend>toggle切换日期的显示隐藏</legend>
                            <input ms-widget="datepicker" data-toggle ="toggle"/>
                            <button ms-click="toggleDatepicker" style="position:absolute;right:0;top: 20px;">toggle calendar</button>
                            <p>说明：可以通过toggle切换datepicker的显示与隐藏，这需要在绑定datepicker时，设置ms-data-toggle属性指向demo controller中的属性</p>
                        </fieldset>
                        <fieldset>
                            <legend>calendarLabel配置日历的说明label</legend>
                            <input ms-widget="datepicker" data-datepicker-calendar-label="选择你的日期" />
                            <p>说明：配置calendarLabel来显示你独特的日历说明</p>
                        </fieldset>
                        <fieldset>
                            <legend>使日历可以通过下拉框切换月份和年份的显示</legend>
                            <input ms-widget="datepicker" data-datepicker-change-month-and-year="true" />
                            <p>说明：配置changeMonthAndYear为true使得用户可以通过下拉选项切换日历的月份和年份</p>
                        </fieldset>
                        <script> 
                            require(["datepicker/avalon.datepicker", "slider/avalon.slider"], function(avalon) {
                                var model = avalon.define("demo", function(vm) {
                                    vm.toggle = false;
                                    vm.toggleDatepicker = function(event) {
                                        model.toggle = !model.toggle;
                                        event.stopPropagation();
                                    }
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
